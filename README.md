# T-Shirts Shop

A fully responsive, multi-page e-commerce website developed by converting a **Figma design** into structured **HTML and CSS**. The project was built using fundamental web technologies only, without any JavaScript, frameworks, or libraries.

**Live Site:** [t-shirtsshopproject07.netlify.app](https://t-shirtsshopproject07.netlify.app)

## Features

- Pixel-accurate implementation of a Figma design
- Fully responsive layout supported at 580px mobile breakpoint
- Product cards with hover-reveal action icons
- Color swatch selectors on product cards
- Active navigation states with hover transitions
- Consistent typography using the Poppins font family across all pages
- Deployed on Netlify

## Getting Started

No dependencies or installations are required.

```bash
git clone https://github.com/GhaZanfar377/T-Shirts-Shop.git
cd T-Shirts-Shop
```

Open `index.html` in any web browser to run the project locally.
